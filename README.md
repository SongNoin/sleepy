## Sleepy 소개
**Sleepy**는 `React-Native` 로 제작된 **수면용품 전문 제품 마켓** `Android 앱`입니다.

또한 `React` 로 제작된 **Sleepy 관리자페이지** `Web` 을 통해 업체에서 상품을 편하게 관리하도록 

제작한 프로젝트입니다.
